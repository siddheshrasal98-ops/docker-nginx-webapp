# Docker Nginx WebApp

## Project Overview

This project demonstrates containerizing a simple static website using Docker and Nginx.

## Technologies

- Docker
- Docker Compose
- Nginx
- HTML

## Features

- Custom Docker image
- Nginx web server
- Static website deployment
- Docker Compose support
